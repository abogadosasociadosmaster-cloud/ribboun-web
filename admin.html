Los cambios se publican en la web guardándolos en GitHub. Configurá una sola vez los datos del repositorio; luego solo tocá "Publicar en la web".

Configuración de GitHub (una sola vez)

El token se crea en GitHub → Settings → Developer settings → Fine-grained tokens, con permiso de lectura/escritura sobre "Contents" del repositorio. Se guarda solo en este navegador.

Si no configuraste GitHub, podés descargar el archivo products.json y subirlo manualmente al repositorio (reemplazando el existente).
